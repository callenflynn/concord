Server Downtime Update

Status update for Concord SMP maintenance and recovery

Published: February 6, 2026

Important Notice

The Concord SMP server experienced downtime from 2/6 to 2/7 while infrastructure updates were completed. Details below.

What Happened?

On 02/06/2026, the server was taken offline to install and configure Proxmox on the host machine. The Minecraft server has been moved into a Debian VM inside Proxmox to improve reliability, isolation, and long-term maintainability.
